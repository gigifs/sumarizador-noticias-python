# 🚀 Sumarização Automática de Notícias em Português

Este projeto demonstra um sistema completo para sumarizar notícias a partir de links da web, comparando técnicas de Processamento de Linguagem Natural (NLP).

## 🎯 Objetivo
O foco é analisar o desempenho de duas abordagens principais:
1. **Extrativa (LexRank):** Seleciona as frases mais importantes do texto original.
2. **Abstrativa (Modelo T5):** Gera um novo texto, reescrevendo as ideias principais.

## 🛠️ Tecnologias Utilizadas
- **Linguagem:** Python
- **Modelos:** T5 (Text-to-Text Transfer Transformer) e LexRank
- **Ambiente:** Google Colab / Jupyter Notebook

## 📊 Estrutura do Projeto
- Configuração do ambiente e instalação de dependências.
- Extração de texto de URLs de notícias.
- Processamento e limpeza de dados (NLP).
- Execução da sumarização extrativa e abstrativa.
- Comparação de resultados.

## 🚀 Como Executar
O projeto está otimizado para execução no **Google Colab**. Basta subir o arquivo `.ipynb` ou exportar para `.py` e rodar as células de configuração.
